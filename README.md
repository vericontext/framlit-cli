# Framlit MCP

MCP (Model Context Protocol) server for Framlit.

## Overview

This project provides an MCP server implementation for the Framlit application.

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```
